# Digital Library Management System

A fullstack Digital Library Management System developed using the MERN Stack. The platform allows students to access books, previous year questions (PYQs), and premium study materials through a userfriendly interface.

## Features

### Student Module
- User Registration
- User Login
- Browse Available Books
- Book Request System
- Access Previous Year Questions (PYQs)
- Access Premium Notes after payment verification
- Student Dashboard

### Admin Module
- Manage Books
- Manage Users
- Approve/Handle Book Requests
- Upload Premium Notes
- Upload PYQs
- Admin Dashboard

## Installation

### Clone Repository

```bash
git clone
https://github.com/abhishek7084187324gupta-commits/digital-library.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
